# PRD — 迭代 1：首页与创建讨论会话

## 功能概述

本迭代实现首页主流程：用户填写议题、选择模板和模型策略、点击"开始讨论"后创建讨论会话并进入讨论页。同时展示最近讨论列表，支持从历史会话直接恢复进入。

本迭代不要求多 Agent 真实生成内容，只要求完成会话创建链路和首页交互闭环。

包含功能点：
1. 首页完整移动端布局
2. 议题输入与字数统计
3. 模板选择
4. 模型策略选择
5. 快速开始议题
6. 最近讨论列表
7. 创建会话 API
8. 查询最近会话 API

---

## Functional Requirements

### 1. 首页布局

**FR-001** [P0] 首页采用移动端竖向单列布局，与原型结构一致，包含：顶部 Hero 区（议题输入、模板卡片、模型策略卡片、开始讨论按钮）、快速开始区、最近讨论区。

- 输入：用户访问首页
- 输出：渲染完整首页，所有区域均正常展示
- 异常：页面加载失败时展示错误状态

---

### 2. 议题输入

**FR-002** [P0] 议题输入框最多接受 100 字，实时展示已输入字数（格式：`n/100`）。

- 输入：用户在输入框中键入内容
- 输出：字数计数器实时更新
- 异常：超出 100 字后不允许继续输入，计数器样式变为警告色

**FR-003** [P0] 点击"开始讨论"时，若议题为空则显示明确错误提示（如行内提示文字），不发起 API 请求。

- 输入：点击"开始讨论"，议题为空
- 输出：输入框下方展示"请输入讨论议题"提示
- 异常：无

---

### 3. 模板选择

**FR-004** [P0] 首页展示模板卡片，默认显示"模板：三国军师团"。点击卡片弹出模板选择 Sheet。

- 输入：点击模板卡片
- 输出：弹出底部 Sheet，展示可选模板列表
- 异常：Sheet 打开失败时不影响主流程

**FR-005** [P0] 模板 Sheet 包含三个选项：三国军师团（默认已选）、创业公司董事会（标注"即将支持"，不可点击选中）、产品辩论桌（标注"即将支持"，不可点击选中）。选择后 Sheet 关闭，卡片显示当前选中模板名。

- 输入：点击"三国军师团"选项
- 输出：Sheet 关闭，模板卡片文字更新为"模板：三国军师团"
- 异常：点击"即将支持"选项时无响应或给出 Toast 提示

---

### 4. 模型策略选择

**FR-006** [P0] 首页展示模型策略卡片，默认显示"模型策略：智能躲避反馈"。点击卡片弹出策略选择 Sheet。

- 输入：点击模型策略卡片
- 输出：弹出底部 Sheet，展示三种策略
- 异常：同 FR-004

**FR-007** [P0] 模型策略 Sheet 包含三个选项：智能躲避反馈（默认，描述"优先快模型，复杂场景切强模型"）、质量优先（描述"更多推理与更强模型，适合严肃决策"）、成本优先（描述"压缩 token 与发言长度，适合快速探索"）。选择后 Sheet 关闭，卡片显示当前选中策略名。

- 输入：点击任意策略选项
- 输出：Sheet 关闭，策略卡片文字更新
- 异常：无

---

### 5. 快速开始

**FR-008** [P1] 首页"快速开始"区域展示 4 个议题 chip，点击后将该议题写入输入框。

预设议题：评估新功能优先级、制定市场进入策略、分析用户增长方案、优化定价策略。

- 输入：点击任意议题 chip
- 输出：输入框内容替换为对应议题文字，字数计数器同步更新
- 异常：无

---

### 6. 最近讨论列表

**FR-009** [P0] 首页"最近讨论"区域展示来自 `GET /api/sessions/recent` 的会话列表，每条记录展示：议题标题、模板名、状态（进行中/已完成）、时间。

- 输入：页面加载
- 输出：异步加载并渲染最近会话列表；列表为空时展示"暂无最近讨论"占位文字
- 异常：接口失败时展示错误提示，不影响首页其他功能

**FR-010** [P0] 点击最近讨论列表中的任意一条记录，跳转至 `/discussion/[sessionId]`。

- 输入：点击列表项
- 输出：路由跳转至对应讨论页
- 异常：sessionId 不存在时讨论页展示错误状态（本迭代不要求完整实现）

---

### 7. 创建会话

**FR-011** [P0] 点击"开始讨论"且议题非空时，调用 `POST /api/sessions` 创建会话。创建成功后跳转至 `/discussion/[sessionId]`（使用 API 返回的 sessionId，首页不自行生成）。

- 输入：topic（用户输入）、templateId（当前选中模板 ID）、modelStrategyId（当前选中策略 ID）
- 输出：跳转至讨论页
- 异常：创建失败时展示 Toast 错误提示，不跳转；提交过程中按钮展示 loading 状态，防止重复提交

---

### 8. 会话创建 API

**FR-012** [P0] `POST /api/sessions` 接收 `{ topic, templateId, modelStrategyId }`，返回 `{ sessionId, topic, template, status, createdAt }`。

- topic 为空时返回错误码 `TOPIC_REQUIRED`
- templateId 对应模板不存在时返回错误码 `TEMPLATE_NOT_FOUND`
- 创建时初始化 `DiscussionState`，phase 置为 `idle`

**FR-013** [P0] `GET /api/sessions/recent` 返回最近会话列表（按创建时间倒序），每条包含 sessionId、topic、templateId、status、createdAt。

---

## Non-Functional Requirements

- **性能**：首页初始渲染（不含最近讨论异步数据）应在 1 秒内完成。
- **安全**：topic 字段服务端进行长度校验（最大 100 字）。
- **兼容性**：移动端 375px 宽度下布局无错位；支持 Chrome / Safari 最新版。

| 编号 | 验收条件 |
|---|---|
| AC-001 | 用户打开首页能看到议题输入框、模板卡片、模型策略卡片、开始讨论按钮、快速开始区、最近讨论列表 |
| AC-002 | 议题输入框最多 100 字，字数实时更新 |
| AC-003 | 点击快速开始 chip 后，输入框填入对应议题 |
| AC-004 | 点击模板卡片能打开模板选择 Sheet，选择后卡片文字更新 |
| AC-005 | 点击模型策略卡片能打开策略选择 Sheet，选择后卡片文字更新 |
| AC-006 | 空议题点击"开始讨论"出现明确错误提示，不发起请求 |
| AC-007 | 非空议题点击"开始讨论"后跳转至 `/discussion/[sessionId]`，sessionId 来自 API 返回值 |
| AC-008 | `POST /api/sessions` 可成功创建会话，返回字段完整 |
| AC-009 | `GET /api/sessions/recent` 可返回最近会话列表 |
| AC-010 | 创建会话时写入 topic、templateId、modelStrategyId，DiscussionState.phase 初始为 `idle` |
| AC-011 | 点击最近讨论列表项可跳转至对应讨论页 |
| AC-012 | 最近讨论列表使用 API 数据，不硬编码 |
| AC-013 | 单元测试覆盖会话创建成功和失败（topic 为空、template 不存在）场景 |
| AC-014 | 页面加载、提交中、错误状态均有对应 UI 展示 |

## Out of Scope
