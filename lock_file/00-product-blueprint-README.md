# 智囊团项目蓝图文档包

本目录包含智囊团项目的总蓝图文档，用于后续迭代开发前的需求评审和技术方案对齐。

## 文件说明

| 文件 | 说明 |
|---|---|
| `00-product-blueprint.md` | 产品立项到正式开发的总蓝图，包含产品定位、功能架构、业务模块、技术架构、数据模型、迭代关系、评审基线和风险控制 |

## 使用方式

后续每个迭代进入开发前，先用 `00-product-blueprint.md` 校验：

1. 本迭代是否符合产品主线。
2. 是否增强多角色讨论体验。
3. 是否破坏 engine / UI / LLM / store / db 的架构边界。
4. 是否具备明确验收标准。
5. 是否需要做设计变更记录。
